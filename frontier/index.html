Questions

Things you probably want to ask before you buy.

Will you see my data?

No. You run it in your own private repo on your own API keys. The product is the framework, not a hosted service. I am a read-only collaborator on the template repository itself, which only contains the framework code. Your instance, your data, your keys.

What does it cost to run?

Roughly £5 to £10 a week on Claude Sonnet at normal scan volumes. One-off targeted briefs typically cost £1 to £2 each in API fees. You pay Anthropic directly; I never touch the invoice.

Can I share it with my team?

Personal-use licence within your organisation. Multiple people in your business can contribute to your instance. No resale, no redistribution, no reproducing the framework as your own product. Full terms in LICENSE.md in the repository.

What if I do not use it?

No refund once collaborator access has been granted. That is why the pre-purchase call exists. Book it if you want to sanity-check fit first.

Do I need to know Python or GitHub Actions?

No Python needed at all. You need enough GitHub familiarity to click through a template, add your API keys as repository secrets, and trigger a workflow manually. If that sounds alien, book the pre-purchase call and I will walk you through what is involved.
